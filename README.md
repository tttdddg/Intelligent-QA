# React + Vite
企业智能问答及数字人交互助手                                                                                            

项目描述：企业智能问答及数字人交互助手是一个集成了智能文档处理和3D数字人交互的企业级应用系统，旨在通过人工智能技术处理多格式文档，构建企业知识库，并提供精准的智能问答服务。同时，利用3D数字人技术提升企业展厅的互动体验，实现个性化讲解和咨询，满足企业内部知识管理和外部形象展示的双重需求。
技术栈：React + TypeScript + Web Speech API + Redux Toolkit + Ant Design
###本项目为大一学生练手原创项目，仅完成了前端架构设计和界面开发，实现文字交互和3D数字人语音交互双模式界面（数字人接口可能挂了）


This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.
